Music Translator for and by Deaf (Alpha)

Welcome to the alpha version of the Music Translator for and by Deaf web app!

This project aims to create a tool that translates musical audio into visual and tactile representations, making music more accessible to the Deaf community.

The back-end of the application is currently in integration testing phase and will be launched soon!

Front-end development will begin after back-end is deployed.

Project Summary:

This project utilizes machine learning models to analyze audio and generate corresponding visual patterns and haptic feedback. The goal is to provide a rich and immersive musical experience for Deaf individuals.

The alpha stage is for testing and gathering feedback. Please report any issues or suggestions on GitHub.
